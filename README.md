# Family Sustainability Plan (Australia - Supply Disruption Scenario)

## Context
This plan assumes:
- Severe disruption to global shipping (e.g., Strait of Hormuz conflict)
- Fuel scarcity impacting domestic transport
- Reduced availability of imported and fresh foods
- Need for cost-controlled, staggered preparation

Target:
- Family of 3 (2 adults, 1 child)
- Minimal budget, incremental buildup
- Food that is:
  - Shelf-stable (6–24 months+)
  - Common in normal cooking
  - Nutritionally balanced

## Strategy Overview
- Build reserves in $200 AUD tranches
- Prioritize calories first, then nutrition diversity
- Use foods that integrate into daily life (no "prepper-only" items)
- Maintain rotation system (eat what you store)

## Caloric Targets
- Adult: ~2,000–2,400 kcal/day
- Child: ~1,200–1,600 kcal/day

Estimated household need:
→ ~5,000 kcal/day

## Monthly Requirement
→ ~150,000 kcal/month
